# Password Generator (Python)

## Description
This is a simple command-line based password generator built using Python. It generates strong and random passwords using a combination of letters, numbers, and special characters.

## Features
- Generate random passwords
- User-defined password length
- Uses letters, digits, and symbols

## Technologies Used
- Python

## How to Run
1. Download or clone the repository
2. Open terminal in the project folder
3. Run the program:
   python password_generator.py

## Example Output
Enter password length: 8  
Generated Password: A@3kL!9q
